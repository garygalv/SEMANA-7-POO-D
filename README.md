# Tarea Semana 7 - Constructores y Destructores en Python

## Descripción
Programa de ejemplo que simula la gestión de conexiones a una base de datos en un sistema de biblioteca.  
Demuestra claramente el uso de:

- `__init__` → inicialización de conexiones simuladas y atributos de estado
- `__del__`  → cierre limpio y ordenado de recursos al destruir el objeto
